Muffins

Combinatorics Algorithms into Code

I was selected by Professor Bill Gasarch to join his project team solving The Muffin Problem, a combinatorics math problem. I worked for two summers to create a public package solving the problem, using a combination of 5 of his algorithms, optimized to solve thousands of muffin problems. I also learned the Julia programming language!

Disease Classification Project

Machine Learning & Disease Prediction

Alongside 4 other engineers, I created a high-precision machine learning model that produced prognoses based on symptoms. I implemented 2 supervised and 1 unsupervised algorithms, KNN, Random Forest and K-Means Clustering, on PCA-reduced data. I also produced a 25-page project finding report and presented a project summary to a Shark Tank panel.

Lyric-Based Genre Classification

Machine Learning & Genre Prediction

I implemented LSTM and NMBA algorithms to create a successful lyric-based classifier alongside my partner. We started by researching and applying the Keras and Seaborn Python packages to a clean dataset. Our final model had a 93% overall accuracy, with up to 97% on genres like hip-hop.

SillyQL

Creating a SQL Interface

This project involved creating a database management platform and implementing common SQL functions for my own version of the interface. The platform was able to create, delete, generate indices, and manipulate rows of a table based on conditionals. This project taught me how to combine multiple data structures to create a larger representation of data!
